Proyecto Final

Instructions

Diseñar y desarrollar un proyecto de Ciencia de Datos aplicando principios metodológicos, conectando múltiples fuentes de datos heterogéneas y comunicando hallazgos de forma clara y reproducible.



Objetivos específicos

1. Formular una pregunta de análisis de datos con valor aplicado.

2. Diseñar o seleccionar una base de datos estructurada en MySQL, PostgreSQL o MongoDB (o combinación).

3. Desarrollar un flujo de extracción, transformación y carga de datos (ETL).

4. Aplicar técnicas de análisis exploratorio de datos para extraer hallazgos relevantes.

5. Documentar todo el proceso de forma clara, reproducible y con enfoque narrativo para publicación en Medium.

El desarrollo del proyecto debe seguir el enfoque CRISP-DM, con etapas claramente diferenciadas:

1. Comprensión del problema

2. Comprensión y conexión a los datos

3. Preparación de los datos

4. Análisis exploratorio (Series de tiempo)

5. (Opcional) Modelado básico

6. Conclusiones accionables
